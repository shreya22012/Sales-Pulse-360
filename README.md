Sales Pulse 360° 

An interactive Sales Analytics Dashboard built using MS Excel.

Features
- City-wise sales analysis
- Sales executive performance tracking
- Target achievement analysis
- Interactive charts and slicers
- KPI-based dashboard

Tools Used
- MS Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

Project Objective
To transform raw sales data into meaningful business insights through interactive dashboard visualization.

Key Learnings
- Data Analytics
- Dashboard Designing
- Business Reporting
- Data Visualization
